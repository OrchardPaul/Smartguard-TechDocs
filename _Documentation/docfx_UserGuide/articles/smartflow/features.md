# Other Features

The Smartflow screen contains optional or configurable features including **fees**, **notes** and **case data**. These can be switched on or off by your Smartflow administrator. All these features are displayed in the right side of the screen as below:

<a href="/images/SmartflowOtherFeatures.jpg" target="_blank" ><img src="/images/SmartflowOtherFeatures.jpg" alt="Smartflow screen features" title="Features" style="width: 350px;" class="shadow"/></a>  
**Figure 1** - Features area of the Smartflow screen.

## Fees
Certain legal actions may require a **fee** or disbursement to be added to the matter e.g. a court fee or a process servers fee. If the Smartflow you are using requires such a fee it will be available in the top right of the screen: 

<a href="/images/SmartflowFeesSection.jpg" target="_blank" ><img src="/images/SmartflowFeesSection.jpg" alt="Fee Section" title="Fee Section" style="width: 350px;" class="shadow"/></a>        
**Figure 2** - Displays all available fees for this Smartflow.

Double clicking the **fee** will result in a pop up where you get the opportunity to amend it before clicking OK to create an electronic posting slip.

Once the slip is created it is automatically sent to accounts for processing. The **fee** can be viewed in the `Posting Slip` area of the `Accounts` tab:

<a href="/images/SmartflowFeeAddedToLedger.jpg" target="_blank" ><img src="/images/SmartflowFeeAddedToLedger.jpg" alt="Posting Slip" title="Posting Slip" style="width: 350px;" class="shadow"/></a>    
**Figure 3** - Displays `Posting Slips` area of the `Accounts` tab.

## Notes
Smartflow allows users to make **notes** which can be viewed by anybody running the Smartflow. To update or delete an existing note simply double click it to open the `Notes` tab. 

> [!TIP]  
> To create a new note, navigate to the **Notes** tab on the `top menu bar` and then click the green `Add New Note` button: 

<a href="/images/SmartflowNotesSection.jpg" target="_blank" ><img src="/images/SmartflowNotesSection.jpg" alt="Notes Section" title="Notes" style="width: 350px;" class="shadow"/></a>  
**Figure 4** - Displays the **notes** area.

## Case Data
Case data can be displayed on the Smartflow screen to assist in decision making and to store useful data e.g. hearing dates. Any amount of **case data** can be displayed over 6 tabs. Some data will be automatically updated but all can be manually popultated or amended by simply double clicking on the required field.

<a href="/images/SmartflowCaseDetailsSection.jpg" target="_blank" ><img src="/images/SmartflowCaseDetailsSection.jpg" alt="Case Data" title="Data" style="width: 350px;" class="shadow"/></a>   
**Figure 5** - Displays all available **case data** for this Smartflow.
