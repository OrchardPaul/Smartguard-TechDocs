# Smartflow Layout

Smartflow has a single interface regardless of worktype. The layout is the same at all times, only the contents of the screen changes depending upon the stage the case is at.

# Accessing Precedent Letters  
Each Smartflow will contain a unique list of precendent letters, documents or workflows to choose from. The list will be configured to provide the user with the relevant options for the current task or milestone.

To process a document simply double click on the required action from the list within the `Next Action`section of the screen. 

> [!TIP]  
> Multiple actions can be selected for processing. A "Y" in the `Run` column indicates which actions have been selected for processing.

## Attachments
Some **actions** may have attachments, these are indicated within the `Att` column. These are usually automatically selected at the same time as the main **action** is selected. To view attachments click on the `Attachments` tab. Actions can be unchecked if not required.

> [!NOTE]  
> This documentation may sometimes refer to **'actions'** within the `Next Action` section of Smartflow. These can be any one of the following; Letter, Document, Form or other Task/Smartflow.

**Figure 3** - Click the `Attachments` tab to view `Attached docments`.

## File History
The file history tab contains all completed letters and documents for the current Smartflow. To view completed letters and documents click on the `Attachments` tab then double click the required action to open it in its' native program.

## Optional Documents
As well as attachments, Smartflow allows you to have `optional documents` which are documents that are sometimes, but not always, sent with the main **action**. To view and select optional documents click the `Optional Documents` toggle situated to the above right of the `Next Action` area (see figure 2).

> [!TIP]  
> Multiple `optional documents` can be selected for processing at any one time.

## Information Help Button
Once **actions** are selected from the `Next Action` area you can click on the `Information` icon <img src="/images/SmartflowSelectedActionInfoIcon.jpg" alt="Info Icon" title="Info Icon"/> to view the action that will be performed by the Smartflow as a consequence of your selection as below.
<a href="/images/SmartflowSelectedActionInfo.jpg" target="_blank" ><img src="/images/SmartflowSelectedActionInfo.jpg" alt="Action Information" title="Action Information" style="width: 250px;" class="shadow"/></a>  
**Figure 4** - Displays all actions about to be undertaken.

> [!TIP]  
> If you do not wish to proceed simply click the green OK button and then deselect the **action** you previously selected.

# Schedule  
When an **action** is selected in the `Next Action` area explained above, Smartflow will usually create a new scheduled task (reminder) which will be assigned to a user for a certain period of time, e.g. 7 days, or for a specific date,    e.g. a hearing date. It will usually also create a history note and amend the status of the Smartflow.
<a href="/images/SmartflowSheduledSection.jpg" target="_blank" ><img src="/images/SmartflowSheduledSection.jpg" alt="Scheduled Task" title="Scheduled Task" style="width: 250px;" class="shadow"/></a>  
**Figure 5** - Displays schedule and history note.

> [!TIP]  
> Double click the `Scheduled Task` to amend the name and date of the scheduled task, change the assignee or delete the task altogether.   

## Messages
**Messages** can be used as a useful reminder or training aid for new or inexperienced users. They are actived when an **action** is selected in the `Next Action` area and are visible at the bottom left of the Smartflow screen.
