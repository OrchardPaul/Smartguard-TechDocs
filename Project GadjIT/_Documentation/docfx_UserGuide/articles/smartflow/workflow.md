# Smartflow Layout

Smartflow has a single interface regardless of worktype. The layout is the same at all times, only the contents of the screen changes depending upon the stage the case is at.

# Accessing Precedent Letters  
Each Smartflow will contain a unique list of precendent letters and document to choose from. The list will be set to provide the user with the relevant options for the current task or milestone.

To process a document simply double click on the required item from the list within the section `Next Action`. 

> [!TIP]  
> Multiple items can be selected for processing. A "Y" in the `Run` column indicates which items have been selected for processing.

## Attachments
Some **items** may have attachemnts, these are indicated within the `Att` column. These are usually automatically selected at the same time as the main **item** is selected. To view attachments click on the `Attachments` tab. Items can be unchecked if not required.

> [!NOTE]  
> This documentation may sometimes refer to **'items'** within the `Next Action` section of Smartflow. These can be any one of the following; Letter, Document, Form or other Task/Smartflow.
